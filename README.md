## Cách chạy trên Visual Studio Code

1. Giải nén file `.zip`.
2. Mở Visual Studio Code.
3. Chọn **File > Open Folder** và mở thư mục `luna-fashion-website`.
4. Cài tiện ích **Live Server** nếu chưa có.
5. Bấm chuột phải vào file `index.html`.
6. Chọn **Open with Live Server**.

Bạn cũng có thể mở trực tiếp file `index.html` bằng trình duyệt Chrome hoặc Edge.
